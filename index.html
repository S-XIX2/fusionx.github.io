📚 Complete Setup Guide

  1. Create GitHub Repository:
    Visit github.com/new and create a public repository
  2. Prepare Your Files:
    Have your kernel ZIP files ready for upload
  3. Create a Release:
    Go to your repo → Releases → Create new release → Add tag → Upload files → Publish
  4. Connect to FusionX:
    Enter your GitHub username and repository name above, then click "Connect Repository"
