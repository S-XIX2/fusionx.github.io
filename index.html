📖 How to Set Up

  1. Create a GitHub Repository:
    • Go to github.com/new
    • Create a repository (e.g., "fusionx-releases")
    • Make it public
  2. Upload Your Files as Releases:
    • Go to your repository
    • Click "Releases" → "Create a new release"
    • Add a tag (e.g., "v1.0")
    • Upload your ZIP files
    • Publish the release
  3. Enter Repository Info:
    • Fill in your GitHub username and repository name above
    • Click "Save Configuration"
    • Your releases will appear automatically!
